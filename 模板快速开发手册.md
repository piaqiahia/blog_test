## 1. 快速入门

### 1.1 模板系统概述

H3Blog 使用 **Jinja2** 模板引擎，采用以下组织结构：

```
app/main/themes/{主题名}/
├── common/           # 公共组件（header、footer、sidebar）
├── macros/           # 可复用的宏（分页、评论、消息提示）
├── static/           # 主题静态资源（CSS、JS、图片）
├── base.html         # 基础模板
├── index.html        # 首页
├── article.html      # 文章详情页
├── category.html     # 分类页
└── tag.html          # 标签页
```

### 1.2 快速创建一个新页面

**步骤 1：创建视图函数** ([app/main/view/](app/main/view/))

```python
from flask import render_template
from app.main.view import build_template_path

@main_bp.route('/about')
def about():
    return render_template(build_template_path('about.html'))
```

**步骤 2：创建模板文件** (`app/main/themes/tend/about.html`)

```jinja2
{% extends 'tend/common/base.html' %}

{% block title %}关于我们 - {{ config.H3BLOG_TITLE }}{% endblock %}

{% block content %}
<div class="container">
    <h1>关于我们</h1>
    <p>这是关于页面内容</p>
</div>
{% endblock %}
```

### 1.3 关键概念

| 概念 | 说明 |
|------|------|
| **模板继承** | 使用 `extends` 继承基础模板，通过 `block` 覆盖内容 |
| **全局函数** | 直接在模板中调用的函数，如 `get_articles()` |
| **过滤器** | 使用 `|` 处理变量，如 `{{ time\|pretty_date }}` |
| **宏组件** | 可复用的模板片段，如分页组件 |

## 2. 核心模板全局函数

### 2.1 文章相关

#### `get_articles()` - 获取文章列表（最常用）

```jinja2
{# 基本用法：获取最新文章 #}
{% set articles = get_articles(is_page=true, page=1, per_page=15) %}

{# 高级用法：按分类和标签过滤 #}
{% set articles = get_articles(
    categorys='python,flask',    # 按分类过滤（支持逗号分隔多个）
    tags='web,backend',          # 按标签过滤
    is_hot=true,                 # 是否热门文章
    hot_num=100,                 # 热门文章阈值（浏览量）
    orderby='desc',              # 排序：'asc'=升序, 'desc'=降序
    is_page=true,                # 开启分页
    page=request.args.get('page', 1),
    per_page=15                  # 每页数量
) %}

{# 遍历文章 #}
{% for article in articles.items %}
    <h3>{{ article.title }}</h3>
    <p>{{ article.summary|truncate(120) }}</p>
    <span>{{ article.publish_time|pretty_date }}</span>
    <span>浏览量: {{ article.vc }}</span>
{% endfor %}
```

#### `get_article_by_id()` - 根据 ID 获取单篇文章

```jinja2
{% set article = get_article_by_id(123) %}
<h2>{{ article.title }}</h2>
```

### 2.2 分类相关

#### `get_categorys()` - 获取分类列表

```jinja2
{# 获取所有可见分类 #}
{% set categories = get_categorys() %}

{# 获取指定分类 #}
{% set categories = get_categorys(names='python,flask') %}

{# 遍历分类 #}
{% for category in categories %}
    <a href="{{ category.url }}">{{ category.title }}</a>
    <span>{{ category.desp }}</span>
{% endfor %}
```

#### `get_category_by_name()` - 获取单个分类

```jinja2
{% set category = get_category_by_name('python') %}
<h2>{{ category.title }}</h2>
<p>{{ category.desp }}</p>
```

#### `get_categorys_by_parent()` - 获取子分类

```jinja2
{# 根据父级 ID 获取子分类 #}
{% set sub_categories = get_categorys_by_parent(parent_id=1) %}

{# 根据父级名称获取子分类 #}
{% set sub_categories = get_categorys_by_parent_name(parent_name='programming') %}
```

#### 根据分类获取文章

```jinja2
{% set category = get_category_by_name('python') %}
{% set articles = category.articles(
    page=request.args.get('page', 1),
    is_page=true,
    per_page=15
) %}
```

### 2.3 标签相关

#### `get_tags()` - 获取标签列表

```jinja2
{# 获取所有标签对象 #}
{% set tags = get_tags() %}

{# 获取指定标签 #}
{% set tags = get_tags(tags='python,flask') %}

{# 只获取标签名称 #}
{% set tag_names = get_tags(rtype='name') %}

{# 标签云显示 #}
{% for tag in tags %}
    <a href="{{ url_for('main.tag', t=tag.code) }}" class="tag">{{ tag.name }}</a>
{% endfor %}
```

### 2.4 Banner 相关

#### `get_banners()` - 获取轮播图

```jinja2
{# 获取顶部轮播图 #}
{% set banners = get_banners(mtype='top') %}

{# 遍历显示 #}
{% for banner in banners %}
    <img src="{{ banner.img }}" alt="{{ banner.title }}">
    <a href="{{ banner.url }}">{{ banner.title }}</a>
{% endfor %}
```

### 2.5 用户信息

#### `current_user` - 当前用户对象

```jinja2
{# 检查用户是否登录 #}
{% if current_user.is_authenticated %}
    <span>欢迎，{{ current_user.username }}</span>
    <a href="{{ url_for('main.logout') }}">退出</a>
{% else %}
    <a href="{{ url_for('main.login') }}">登录</a>
    <a href="{{ url_for('main.regist') }}">注册</a>
{% endif %}

{# 检查用户类型 #}
{% if current_user.user_type == 'vip' %}
    <div class="vip-content">VIP 专属内容</div>
{% endif %}

{# 检查是否为管理员 #}
{% if current_user.is_admin or current_user.username == 'admin' %}
    <a href="{{ url_for('admin.index') }}">进入管理后台</a>
{% endif %}
```

#### `has_perms()` - 权限检查

```jinja2
{% if has_perms('article:edit') %}
    <button>编辑文章</button>
{% endif %}

{% if has_perms('user:delete') %}
    <a href="{{ url_for('main.user_delete', id=user.id) }}">删除</a>
{% endif %}
```

## 3. 常用过滤器

### 3.1 `pretty_date` - 美化日期显示

```jinja2
{# 相对时间显示 #}
{{ article.publish_time|pretty_date }}
{# 输出示例：2小时前、3天前、1周前 #}

{{ article.timestamp|pretty_date }}
```

### 3.2 `hidden_content` - 隐藏内容处理

```jinja2
{# 未登录用户看到登录提示，已登录用户看到完整内容 #}
<div class="article-content">
    {{ article.content_html|hidden_content|safe }}
</div>
```

### 3.3 `truncate` - 文本截断

```jinja2
{# 截断为指定长度 #}
{{ article.summary|truncate(120) }}
{{ article.title|truncate(50, killwords=True) }}
```

### 3.4 `seconds_hms` - 秒数转时分秒

```jinja2
{{ 3665|seconds_hms }}
{# 输出: 1:1:5 #}
```

## 4. 分页功能

### 4.1 导入分页宏

```jinja2
{% import "tend/macros/_patination.html" as page_macros %}
```

### 4.2 获取分页数据

```jinja2
{# 方式1：使用 get_articles 函数 #}
{% set articles = get_articles(
    is_page=true,
    page=request.args.get('page', 1),
    per_page=15
) %}

{# 方式2：使用分类的 articles 方法 #}
{% set articles = category.articles(
    page=request.args.get('page', 1),
    is_page=true,
    per_page=15
) %}
```

### 4.3 遍历分页数据

```jinja2
{% for article in articles.items %}
    <article>
        <h2>{{ article.title }}</h2>
        <p>{{ article.summary }}</p>
    </article>
{% endfor %}

{# 分页信息 #}
<p>第 {{ articles.page }} 页，共 {{ articles.pages }} 页，总计 {{ articles.total }} 条</p>
```

### 4.4 显示分页导航

```jinja2
{# 基本用法 #}
{{ page_macros.pagination_widget(articles, '.index') }}

{# 带参数用法 #}
{{ page_macros.pagination_widget(articles, '.category', c=category.name) }}

{# 带多个参数 #}
{{ page_macros.pagination_widget(articles, '.index', st='hot', category='python') }}
```

## 5. 模板继承模式

### 5.1 基础模板结构 ([base.html](app/main/themes/tend/common/base.html))

```jinja2
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">

    {# SEO 信息 #}
    <title>{% block title %}{{ config.H3BLOG_TITLE }}{% endblock %}</title>
    <meta name="description" content="{% block description %}{{ config.H3BLOG_DESCRIPTION }}{% endblock %}">
    <meta name="keywords" content="{% block keywords %}{{ config.H3BLOG_KEYWORDS }}{% endblock %}">

    {# 自定义 meta #}
    {% block meta %}{% endblock %}

    {# CSRF 令牌 #}
    <meta name="csrf-token" content="{{ csrf_token() }}">

    {# CSS 样式 #}
    {% block css %}
    <link href="{{ url_for('main.static', filename='css/base.css') }}" rel="stylesheet">
    {% endblock %}
</head>

<body>
    {# 头部 #}
    {% include 'tend/common/header.html' %}

    <main>
        {# 主内容区 #}
        {% block main %}
        <div class="container">
            <div class="row">
                <div class="col-lg-8">
                    {% block content %}{% endblock %}
                </div>
                <div class="col-lg-4">
                    {% include 'tend/common/sidebar.html' %}
                </div>
            </div>
        </div>
        {% endblock %}
    </main>

    {# 页脚 #}
    {% include 'tend/common/footer.html' %}

    {# JavaScript #}
    {% block js %}
    <script src="{{ url_for('main.static', filename='js/base.js') }}"></script>
    {% endblock %}
</body>
</html>
```

### 5.2 子模板实现

```jinja2
{% extends 'tend/common/base.html' %}

{# 覆盖 title #}
{% block title %}{{ article.title }} - {{ super() }}{% endblock %}

{# 添加额外 CSS #}
{% block css %}
{{ super() }}
<link href="{{ url_for('main.static', filename='css/article.css') }}" rel="stylesheet">
{% endblock %}

{# 覆盖内容区 #}
{% block content %}
<div class="article-detail">
    <h1>{{ article.title }}</h1>
    <div class="content">
        {{ article.content_html|safe }}
    </div>
</div>
{% endblock %}

{# 添加额外 JS #}
{% block js %}
{{ super() }}
<script src="{{ url_for('main.static', filename='js/article.js') }}"></script>
{% endblock %}
```

## 6. 常见页面模板示例

### 6.1 首页（文章列表）

```jinja2
{% extends 'tend/common/base.html' %}
{% import "tend/macros/_patination.html" as page_macros %}
{% import "tend/macros/_flash.html" as flash_macros %}

{% block content %}
{{ flash_macros.flash_msg() }}

{# 轮播图 #}
{% set banners = get_banners(mtype='top') %}
{% if banners|length > 0 %}
<div id="carousel" class="carousel slide" data-ride="carousel">
    <div class="carousel-inner">
        {% for banner in banners %}
        <div class="carousel-item {{ 'active' if loop.first else '' }}">
            <a href="{{ banner.url }}">
                <img src="{{ banner.img }}" alt="{{ banner.title }}">
            </a>
        </div>
        {% endfor %}
    </div>
</div>
{% endif %}

{# 排序切换 #}
<div class="sort-bar">
    <a class="{{ 'active' if request.args.get('st','time') == 'time' else '' }}"
       href="{{ url_for('.index', st='time') }}">时间排序</a>
    <a class="{{ 'active' if request.args.get('st','time') == 'hot' else '' }}"
       href="{{ url_for('.index', st='hot') }}">热度排序</a>
</div>

{# 文章列表 #}
{% set st = request.args.get('st','time') %}
{% set articles = get_articles(
    page=request.args.get('page', 1),
    is_page=true,
    is_hot=(st == 'hot'),
    per_page=15
) %}

{% for article in articles.items %}
<div class="article-item">
    <img src="{{ article.thumbnail_img() }}" alt="{{ article.title }}">
    <h3><a href="{{ url_for('main.article', name=article.name) }}">{{ article.title }}</a></h3>
    <p>{{ article.summary|truncate(120) }}</p>
    <div class="meta">
        <span>{{ article.publish_time|pretty_date }}</span>
        <span>{{ article.vc }} 阅读</span>
        <a href="{{ url_for('main.category', c=article.category.name) }}">
            {{ article.category.title }}
        </a>
    </div>
</div>
{% endfor %}

{# 分页 #}
<div class="pagination">
    {{ page_macros.pagination_widget(articles, '.index', st=st) }}
</div>
{% endblock %}
```

## 8. 常用模式和技巧

### 8.1 循环中的条件判断

```jinja2
{# 标签列表，用 • 分隔 #}
{% for tag in article.tags.all() %}
    <a href="{{ url_for('main.tag', t=tag.code) }}">{{ tag.name }}</a>
    {% if not loop.last %}
        <span>•</span>
    {% endif %}
{% endfor %}

{# 判断是否为第一个/最后一个 #}
{% for banner in banners %}
    <div class="item {{ 'active' if loop.first else '' }}">
        {{ banner.title }}
    </div>
{% endfor %}
```

### 8.2 空值处理

```jinja2
{# 检查文章列表是否为空 #}
{% if articles and articles.items %}
    {% for article in articles.items %}
        {# 显示文章 #}
    {% endfor %}
{% else %}
    <div class="empty-state">
        <p>暂无数据</p>
    </div>
{% endif %}

{# 检查字段是否存在 #}
{% if article.summary %}
    <p>{{ article.summary }}</p>
{% endif %}

{# 提供默认值 #}
{{ article.desp or '暂无描述' }}
```

### 8.3 URL 参数处理

```jinja2
{# 获取 URL 参数 #}
{% set page = request.args.get('page', 1) %}
{% set st = request.args.get('st', 'time') %}
{% set category = request.args.get('c', '') %}

{# 构建带参数的 URL #}
<a href="{{ url_for('.index', st='hot') }}">热度排序</a>
<a href="{{ url_for('.index', st='time', page=1) }}">时间排序</a>

{# 保留当前参数 #}
<a href="{{ url_for('.index', st=request.args.get('st','time'), page=2) }}">第2页</a>
```

### 8.4 消息提示

```jinja2
{# 导入消息宏 #}

{# 显示消息（成功、错误、警告、信息）#}

```

### 8.5 条件渲染

```jinja2
{# 响应式显示 #}
<p class="d-none d-sm-block">{{ article.summary|truncate(120) }}</p>
<p class="d-block d-sm-none">{{ article.summary|truncate(64) }}</p>

{# 根据用户类型 #}
{% if current_user.user_type == 'vip' %}
    <div class="vip-badge">VIP 用户</div>
{% elif current_user.is_authenticated %}
    <div class="member-badge">普通用户</div>
{% endif %}
```

## 9. 最佳实践

### 9.1 模板设计原则

1. **使用模板全局函数** - 减少视图层的业务逻辑
2. **合理使用模板继承** - 避免代码重复，保持 DRY 原则
3. **使用宏组件** - 提高代码复用性（分页、评论等）
4. **注意 XSS 防护** - 使用 `|safe` 过滤器时要谨慎
5. **SEO 优化** - 正确设置 title、keywords、description
6. **性能优化** - 避免在模板中执行复杂查询

### 9.2 安全性

```jinja2
{# ✅ 正确：使用 CSRF 令牌 #}
<form method="POST">
    <input type="hidden" name="csrf_token" value="{{ csrf_token() }}">
    <!-- 表单字段 -->
</form>

{# ⚠️ 谨慎使用 safe 过滤器 #}
{# 只有确认内容安全时才使用 #}
{{ article.content_html|safe }}

{# ❌ 不要这样做 #}
{{ user_input|safe }}  {# 可能导致 XSS 攻击 #}
```

### 9.3 性能优化

```jinja2
{# ✅ 好：使用模板全局函数（已缓存）#}
{% set articles = get_articles(is_page=true, page=1) %}

{# ❌ 避免：在模板中执行复杂查询 #}
{% for article in Article.query.filter(...).all() %}
    ...
{% endfor %}

{# ✅ 好：使用 limit 限制数量 #}
{% set hot_articles = get_articles(is_hot=true, per_page=5) %}
```

### 9.4 可维护性

```jinja2
{# ✅ 好：使用有意义的变量名 #}
{% set latest_articles = get_articles(orderby='desc', per_page=10) %}

{# ❌ 避免：使用无意义的变量名 #}
{% set a = get_articles() %}
{% set d = category.desp %}

{# ✅ 好：添加注释 #}
{# 获取 Python 分类下的热门文章 #}
{% set python_articles = get_articles(categorys='python', is_hot=true) %}
```
