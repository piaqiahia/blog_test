## 1. 模板全局函数

### 1.1 时间相关函数

#### `now()`
- **功能描述**: 获取当前时间
- **返回值类型**: `datetime`
- **使用示例**:
```jinja2
当前时间: {{ now() }}
当前日期: {{ now().strftime('%Y-%m-%d') }}
```

#### `strptime(t, format='%Y-%m-%d')`
- **功能描述**: 字符串转日期对象
- **参数说明**:
  - `t`: 日期字符串
  - `format`: 日期格式，默认为 `%Y-%m-%d`
- **返回值类型**: `datetime`
- **使用示例**:
```jinja2
{% set date_obj = strptime('2023-12-01') %}
格式化日期: {{ date_obj.strftime('%Y年%m月%d日') }}
```

#### `get_website_run_time(start_time_str)`
- **功能描述**: 计算网站运行时间
- **参数说明**:
  - `start_time_str`: 网站开始运行时间字符串
- **返回值类型**: `str`
- **使用示例**:
```jinja2
网站已运行: {{ get_website_run_time('2020-02-20') }}
```

### 1.2 数据查询函数

#### `get_articles()`
- **功能描述**: 根据条件获取已发布的文章
- **参数说明**:
  - `categorys`: 文章分类标识（逗号分割，如 "python,flask,django"）
  - `tags`: 文章标签（逗号分割，如 "python,安全,何三笔记"）
  - `is_hot`: 是否热门文章（根据浏览量）
  - `hot_num`: 热门文章阈值
  - `orderby`: 排序方式（"asc"=升序，"desc"=降序）
  - `is_page`: 是否分页
  - `page`: 分页页数
  - `per_page`: 每页条数
- **返回值类型**: `List[Article]` 或 `Paginate`
- **使用示例**:
```jinja2
{# 获取Python分类的文章，分页显示 #}
{% set articles = get_articles(categorys='python', is_page=true, page=request.args.get('page',1), per_page=10) %}

{# 获取热门文章 #}
{% set hot_articles = get_articles(is_hot=true, hot_num=100, per_page=5) %}
```

#### `get_article_by_id(id)`
- **功能描述**: 根据ID获取文章
- **参数说明**:
  - `id`: 文章ID
- **返回值类型**: `Article`
- **使用示例**:
```jinja2
{% set article = get_article_by_id(123) %}
```

#### `get_categorys(names=None, visible=1)`
- **功能描述**: 获取文章分类
- **参数说明**:
  - `names`: 分类名称（逗号分割）
  - `visible`: 是否可见（1=可见，0=隐藏）
- **返回值类型**: `List[Category]`
- **使用示例**:
```jinja2
{# 获取所有可见分类 #}
{% set categories = get_categorys() %}

{# 获取指定分类 #}
{% set python_cats = get_categorys(names='python,flask') %}
```

#### `get_category_by_name(name)`
- **功能描述**: 根据名称获取分类
- **参数说明**:
  - `name`: 分类名称
- **返回值类型**: `Category`

#### `get_categorys_by_parent(parent_id=None, visible=1)`

- **参数说明**:

- **返回值类型**: `List[Category]`
- **使用示例**:
```jinja2

```

#### `get_categorys_by_parent_name(parent_name=None, visible=1)`
- **功能描述**: 根据父级名称获取子分类
- **参数说明**:

- **使用示例**:

#### `get_tags(tags=None, rtype='obj')`
- **功能描述**: 获取系统标签
- **参数说明**:

### 1.4 权限相关函数

## 2. 模板过滤器

### 2.1 时间格式化过滤器

#### `seconds_hms`
- **功能描述**: 将秒数转换为时分秒格式
- **输入类型**: `int`
- **输出类型**: `str`
- **使用示例**:
```jinja2
{% set duration = 3665 %}
时长: {{ duration|seconds_hms }}
{# 输出: 1:1:5 #}
```

#### `pretty_date`
- **功能描述**: 美化日期显示（相对时间）
- **输入类型**: `datetime`
- **输出类型**: `str`
- **使用示例**:
```jinja2
发布时间: {{ article.publish_time|pretty_date }}
{# 可能输出: 2小时前、3天前等 #}
```

### 2.2 内容处理过滤器

#### `hidden_content`
- **功能描述**: 处理隐藏内容（需要登录才能查看）
- **输入类型**: `str`
- **输出类型**: `str`
- **使用示例**:
```jinja2
{{ article.content_html|hidden_content|safe }}
```

#### `request_page_params`
- **功能描述**: 处理分页参数，移除page和指定参数
- **输入类型**: `dict`
- **输出类型**: `dict`
- **使用示例**:
```jinja2
{% set params = request.args.to_dict()|request_page_params('st') %}
```

<p>发布时间: {{ article.publish_time|pretty_date }}</p>

## 5. 分页组件实现

### 5.1 分页宏组件 (_patination.html)

#### 核心参数
- `pagination`: 分页对象
- `endpoint`: 路由端点
- `**kwargs`: 额外URL参数

#### 实现代码
```jinja2
{% macro pagination_widget(pagination, endpoint) %}
<ul class="pagination">
    {# 上一页 #}
    <li {% if not pagination.has_prev %}class="page-item disabled"{% endif %}>
        <a class="page-link" href="{% if pagination.has_prev %}{{ url_for(endpoint, page=pagination.page - 1, **kwargs) }}{% else %}#{% endif %}">
            <span>&laquo;</span>
        </a>
    </li>
    
    {# 页码 #}
    {% for p in pagination.iter_pages() %}
        {% if p %}
            {% if p == pagination.page %}
            <li class="page-item active"><a class="page-link" href="{{ url_for(endpoint, page=p, **kwargs) }}">{{ p }}</a></li>
            {% else %}
            <li class="page-item"><a class="page-link" href="{{ url_for(endpoint, page=p, **kwargs) }}">{{ p }}</a></li>
            {% endif %}
        {% else %}
        <li class="page-item disabled"><a class="page-link" href="#">&hellip;</a></li>
        {% endif %}
    {% endfor %}
    
    {# 下一页 #}
    <li {% if not pagination.has_next %}class="page-item disabled"{% endif %}>
        <a class="page-link" href="{% if pagination.has_next %}{{ url_for(endpoint, page=pagination.page + 1, **kwargs) }}{% else %}#{% endif %}">
            <span>&raquo;</span>
        </a>
    </li>
</ul>
{% endmacro %}
```

#### 使用方式
```jinja2
{% import "tend/macros/_patination.html" as page_macros %}

{# 基本用法 #}
{{ page_macros.pagination_widget(articles, '.index') }}

{# 带参数用法 #}
{{ page_macros.pagination_widget(articles, '.category', c=category.name, st=request.args.get('st','time')) }}
```

## 6. 静态资源引用规则

### 6.1 静态资源存放路径

#### 主题静态资源
- 路径: `app/main/themes/{主题名}/static/`
- 示例: `app/main/themes/tend/static/css/style.css`

#### 公共静态资源
- 路径: `app/static/`
- 示例: `app/static/img/logo.png`

### 6.2 模板中引用方式

#### CSS文件引用
```jinja2
{# 主题CSS #}
<link href="{{ url_for('main.static', filename='css/style.css') }}" rel="stylesheet">

{# 公共CSS #}
<link href="{{ url_for('static', filename='css/common.css') }}" rel="stylesheet">

```

#### JavaScript文件引用
```jinja2
{# 主题JS #}
<script src="{{ url_for('main.static', filename='js/main.js') }}"></script>

{# 公共JS #}
<script src="{{ url_for('static', filename='js/common.js') }}"></script>

```

#### 图片资源引用
```jinja2
{# 主题图片 #}
<img src="{{ url_for('main.static', filename='img/logo.png') }}" alt="Logo">

{# 公共图片 #}
<img src="{{ url_for('static', filename='img/bg.jpg') }}" alt="Background">

{# 文章缩略图 #}
<img src="{{ article.thumbnail_img() }}" alt="{{ article.title }}">

{# 上传的图片 #}
<img src="/uploads/{{ image_path }}" alt="Uploaded Image">
```

### 6.3 资源引用最佳实践

#### 使用url_for函数
```jinja2
{# 推荐 ✅ #}
<link href="{{ url_for('main.static', filename='css/style.css') }}" rel="stylesheet">

{# 不推荐 ❌ #}
<link href="/static/css/style.css" rel="stylesheet">
```

#### 条件加载资源
```jinja2
{% if config.DEBUG %}
    {# 开发环境使用未压缩版本 #}
    <script src="{{ url_for('main.static', filename='js/dev.js') }}"></script>
{% else %}
    {# 生产环境使用压缩版本 #}
    <script src="{{ url_for('main.static', filename='js/prod.min.js') }}"></script>
{% endif %}
```

#### 资源版本控制
```jinja2
{# 添加版本号避免缓存 #}
<link href="{{ url_for('main.static', filename='css/style.css') }}?v=1.0.0" rel="stylesheet">
```

## 8. 常见问题与解决方案

### 8.1 模板函数调用失败
- **问题**: 模板函数未定义或调用失败
- **解决方案**: 检查函数是否在 `template_global.py` 中正确定义

### 8.2 分页显示异常
- **问题**: 分页链接不正确或页码显示错误
- **解决方案**: 确保分页对象正确传递，检查URL参数处理

### 8.3 静态资源404
- **问题**: 静态资源无法加载
- **解决方案**: 检查资源路径是否正确，确认文件是否存在

### 8.4 权限判断错误
- **问题**: 权限判断逻辑不正确
- **解决方案**: 使用 `has_perms()` 和 `check_perms()` 函数进行权限验证
